# Billing & Inventory Management System

A web-based application for managing products, billing, suppliers, and reports.
This system helps businesses track inventory, generate invoices, and monitor sales efficiently.

---

## Features

* Dashboard with key statistics
* Product / Inventory management
* Billing system
* AI Stock Prediction
* Reports
* Customer management

---

## Tech Stack

Frontend:

* React.js
* CSS

Backend:

* Node.js
* Express.js

Database:

* MySQL

## Installation

Clone the repository

```

```

```
npm install
```

```
npm run dev
```
